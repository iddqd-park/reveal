# REVEAL

[English](README.md)

**Project REVEAL(리빌)**은 친구들이 보낸 '의심스러운 링크'를 안전하게 확인하기 위한 도구입니다. 낚시성 이미지나 후방주의(NSFW) 사이트가 의심될 때, 강력한 블러(흐림) 커튼 뒤에서 먼저 내용을 간략히 파악하고, 마음의 준비가 되면 슬라이더를 조작해 천천히 내용을 확인하세요.

## Our Philosophy
IDDQD Internet은 별도의 DB나 회원가입 없이, 순수 HTML/JS만으로 브라우저에서 즉시 실행되는 도구를 개발합니다. AI 기능을 제공할 때도 데이터 상태를 유지하지 않으며(stateless), 어떠한 기록도 남기지 않는 원칙을 고수합니다.

## 주요 기능
-   **안전한 공유**: 특정 콘텐츠가 자동으로 로드되는 공유 링크 생성.
-   **프라이버시 커튼**: 콘텐츠는 초기에 강력한 블러 처리가 된 상태로 로드됩니다.
-   **인터랙티브 리빌**: 슬라이더를 움직여 블러를 걷어내고 내용을 확인할 수 있습니다.
-   **기록 없음**: 서버에 어떠한 기록도 남지 않으며, 계정이 필요 없습니다.

## 사용법
1.  **URL 입력**: 공유하고 싶은 웹사이트 주소를 입력하세요.
2.  **로드**: 'LOAD' 버튼을 눌러 사이트를 불러옵니다.
3.  **공유**: 브라우저 주소창의 URL을 복사하세요 (공유 링크가 자동으로 생성됩니다).
4.  **리빌**: 친구에게 링크를 보내세요. 친구는 슬라이더를 조작해 사이트를 확인할 수 있습니다.
5.  **이동**: 'GO' 버튼을 누르면 해당 페이지로 직접 이동합니다.
6.  **리셋**: 홈 아이콘을 누르면 초기 상태로 돌아갑니다.

## 기술 스택
-   **Core**: HTML5, CSS3, Vanilla JavaScript
-   **UI/UX**: Bootstrap 5, 커스텀 글래스모피즘(Glassmorphism) CSS
-   **Audio**: Web Audio API (실시간 비프음 생성)
-   **Platform**: Nginx, Ubuntu Linux
