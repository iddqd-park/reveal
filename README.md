# REVEAL

[한국어](README.ko.md)

**Project REVEAL** is a privacy-focused web tool that allows for the secure and dramatic sharing of URLs. It loads a target website behind a heavy blur curtain, which the user can manually reveal using a slider control.

## Our Philosophy
IDDQD Internet builds zero-DB, zero-signup tools powered by pure HTML/JS for instant browser execution. Even with AI features, we keep it stateless and record-free.

## Features
- **Secure Sharing**: Create shareable links that auto-load specific content.
- **Privacy Curtain**: Content is initially hidden behind a configurable blur layer.
- **Interactive Reveal**: Use the slider to clear the blur and view the content.
- **Zero-Footprint**: No history saved on server, no accounts required.

## Usage
1.  **Enter URL**: Type the website address you want to share.
2.  **Load**: Click 'LOAD' to fetch the site.
3.  **Share**: Copy the URL from your browser's address bar (it now contains the target link).
4.  **Reveal**: Send to a friend. They can use the slider to reveal the site.
5.  **Go**: Click 'GO' to navigate directly to the revealed page.
6.  **Reset**: Click the Home icon to reset everything.

## Tech Stack
-   **Core**: HTML5, CSS3, Vanilla JavaScript
-   **UI/UX**: Bootstrap 5, Custom Glassmorphism CSS
-   **Audio**: Web Audio API (real-time generated beeps)
-   **Platform**: Nginx, Ubuntu Linux
